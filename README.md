# Notfall-Hilfe

Eine einfache Notfall-Webanwendung für GitHub Pages.

## Funktionen

- Standortbestimmung über den Browser
- Anzeige des eigenen Standorts auf einer OpenStreetMap-Karte
- Responsive Oberfläche für Smartphones und Desktop
- Grundlage für weitere Notfall-Funktionen

## Geplante Funktionen

- Krankenhäuser in der Nähe
- Öffentliche Toiletten in der Nähe
- Restaurants in der Nähe
- Apotheken und Polizeistationen
- Wetterwarnungen
- Nachrichtenfeed
- Offline-Unterstützung

## Verwendete Technologien

- HTML
- CSS
- JavaScript
- Leaflet
- OpenStreetMap

## GitHub Pages

Nach dem Aktivieren von GitHub Pages unter Settings → Pages kann die Anwendung direkt im Browser genutzt werden.

## Hinweis

Für die Standortfunktion muss der Nutzer den Zugriff auf seinen Standort erlauben.
